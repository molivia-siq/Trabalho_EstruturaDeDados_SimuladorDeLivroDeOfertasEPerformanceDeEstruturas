# Trabalho_EstruturaDeDados_SimuladorDeLivroDeOfertasEPerformanceDeEstruturas
Projeto acadêmico desenvolvido para aplicação de estruturas de dados lineares (listas encadeadas, pilhas e filas) na construção de um motor de negociação financeira, com foco em análise assintótica e comparação de desempenho em cenários de grande volume de dados.
